## ReIm

```
ReIm(z)
```

> returns a list of the real and imaginary component of the complex number `z`.
 
See
* [Wikipedia - Complex number](https://en.wikipedia.org/wiki/Complex_number)

### Examples

```
>> ReIm(3+4*I)
{3,4}

>> ReIm(0.5 + 2.3*I)
{0.5,2.3}
```
