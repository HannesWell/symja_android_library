## RealSign

```
RealSign(x)
```

> gives `-1`, `0` or `1` depending on whether `x` is negative, zero or positive. For complex number arguments the function will be left unevaluated.
 
See
* [Wikipedia - Sign function](https://en.wikipedia.org/wiki/Sign_function)

### Examples

```
>> RealSign(-2.5)
-1

>> RealSign(-Infinity)
-1
```
