## ListLogLogPlot

```
ListLogLogPlot( { list-of-points } )  
```

> generate an image of a logarithmic X and logarithmic Y plot for the `list-of-points`.
	 
### Examples


```
>> ListLogLogPlot(RandomReal(1, 20)^5)
```
