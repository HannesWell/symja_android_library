## LogIntegral

```
LogIntegral(expr)
```

> returns the integral logarithm of `expr`.
  
See
* [Wikipedia - Logarithmic integral function](https://en.wikipedia.org/wiki/Logarithmic_integral_function)

### Examples

```
>> LogIntegral(0)
0
```
