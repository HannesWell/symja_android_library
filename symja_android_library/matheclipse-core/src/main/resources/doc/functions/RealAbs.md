## RealAbs

```
RealAbs(x)
```

> returns the absolute value of the real number `x`. For complex number arguments the function will be left unevaluated.
  

See:
* [Wikipedia - Absolute value](http://en.wikipedia.org/wiki/Absolute_value)
 

### Examples

```
>> RealAbs(-3)
3

>> RealAbs(-Infinity)
Infinity
```
