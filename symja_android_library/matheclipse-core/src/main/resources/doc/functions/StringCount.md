## StringCount

```
StringCount(string, pattern)
```

> counts all occurences of `pattern` in `string`.
  
### Examples

```
>> StringCount("a#ä_123", WordCharacter)
5

>> StringCount("a#ä_123", LetterCharacter) 
2 
```
